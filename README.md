# Movie Night Invitation

An interactive online invitation for movie night with a tricky "decline" button that randomizes its position until the user clicks "yes", followed by a confetti animation.

## How to Use

1. Open `index.html` in a web browser.
2. Customize the date, time, and location in the HTML file.
3. Share the link with your friends!

## Features

- Hovering or clicking "No" moves the button to a random position.
- Clicking "Yes" triggers a confetti animation and shows a confirmation message.

## Dependencies

- [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti) for the confetti effect (loaded via CDN).
